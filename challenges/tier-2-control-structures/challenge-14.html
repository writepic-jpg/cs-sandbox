Traffic Lights

Ask the user to enter a traffic light colour. If it is 'red', output 'Stop.'. If it is 'amber', output 'Get Ready.'. If it is 'green', output 'Go.'. Watch out for distractors and bin the incorrect lines!

📋

Examiner Marking Notes

  • Distractors: Watch out for sneaky syntax errors or incorrect keywords like PRINT instead of OUTPUT. Bin (🗑️) blocks that are incorrect!
  • Multi-way Keywords: In OCR ERL, ELSEIF is a single, combined keyword. Separating it into two words will result in an automatic syntax error on your Paper 2 exam.
  • Assignment vs Equality: Remember that a single = assigns a value, while a double == compares values inside selection statements.
