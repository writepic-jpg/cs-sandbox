Examiner Marking Notes

  • Integer Division: DIV is the correct ERL operator for finding the number of whole coins, while MOD finds the remainder. Using / would result in a real number, which is a logic error in this context.
  • Concatenation: Combine data and strings inside outputs smoothly using a addition symbol (+). Dot operators (.) are not part of standard J277 syntax rules.
