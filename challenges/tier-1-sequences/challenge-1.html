Examiner Marking Notes

  • Variable Assignment: Ensure the identifier (name) is placed strictly on the left of the assignment operator.
  • Function Nesting: Remember that input() must capture the stream before print() can reference it.
  • Common Pitfall: Missing quotation marks inside the user prompt string argument is a frequent loss of marks on paper exams.
