Examiner Marking Notes

  • Assignment Rules: Arithmetic operations must be calculated on the right side of the expression and stored into the singular variable receiver on the left.
  • Arithmetic Operators: Standard math expressions use standard computer layouts (+, -, *, /). Written English operators like PLUS are completely syntax-invalid.
  • Data Type Awareness: In live code, input functions default to strings. Remember that true mathematical summation requires cast conversions to avoid appending strings together.
