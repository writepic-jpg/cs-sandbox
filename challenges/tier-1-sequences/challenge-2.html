Echo Name

Take a user's first name as a dynamic interactive input, and then print a welcoming output string that echoes that explicit variable back to the terminal layout view.

📋

Examiner Marking Notes

  • Sequence Order: Variables must be declared and stored with input() before they can be printed out. If elements are used beforehand, it throws a runtime NameError.
  • Concatenation vs. Separation: Passing multiple sequential parameters to separate print operations will display strings on entirely fresh output streams.
  • Common Pitfall: Do not confuse string text values "firstname" with identifiers/variables firstname inside the print statement arguments.
